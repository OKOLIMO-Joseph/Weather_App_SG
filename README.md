# Weather_App_SG

A modern, responsive weather application that fetches live weather data from the OpenWeatherMap API. Enter any city name to get current temperature, humidity, wind speed, weather conditions, and dynamic weather icons.

##  Features

-  **City Search** - Search weather conditions for any city worldwide
-  **Real-time Metrics** - Temperature, feels-like temperature, humidity, and wind speed
-  **Dynamic Weather Icons** - Visual weather representation based on current conditions
-  **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
-  **Fast & Lightweight** - Minimal dependencies, optimized performance
-  **Modern Glassmorphic UI** - Beautiful translucent design with smooth animations

##  Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations, Backdrop-filter)
- JavaScript (ES6+)
- OpenWeatherMap API

##  How to Use

1. Clone the repository
2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Replace `YOUR_API_KEY_HERE` in the JavaScript with your actual API key
4. Open `index.html` in your browser
5. Search for any city to view current weather conditions

##  API Reference

This project uses the OpenWeatherMap Current Weather Data API:
- Endpoint: `api.openweathermap.org/data/2.5/weather`
- Parameters: `q={city name}&appid={API key}`
